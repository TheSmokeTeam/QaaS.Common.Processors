# QaaS.Common.Processors

The `QaaS.Common.Processors` package provides common transaction processor hooks that can be used by `QaaS.Mocker` stub configurations.

> Written In C# 14 & .net 10

## Projects

### Functional Projects

* `QaaS.Common.Processors` - Common transaction processors and their configurations.

### Tests

* `QaaS.Common.Processors.Tests` - NUnit test project for common processors.
